> 面向：运营 / 非工程师  
> 目标：5 分钟内独立完成后台基础功能验证  
> 前提：已有运营账号密码（由负责人单独告知，**不写在任何文档或聊天记录里**）

## 1. 打开后台 / 登录

**地址：** `https://hugme2.com/admin/login`

1. 在浏览器地址栏输入上述地址，按 Enter。
2. 输入用户名和密码，点击 **登录**。
3. 登录成功后自动跳转到会话总览页。

**期望现象：**

- 页面顶部显示 `ERIS 运营后台`，左上角有紫色 ERIS 字样。
- 右上角显示你的账号名和角色（如 `admin`）。
- 主区域显示会话列表（有数据时）或「暂无会话」。

---

## 2. 查看会话列表

1. 登录后即进入会话总览页（`/admin`）。
2. 页面显示所有用户会话，每行包含：用户昵称、渠道、状态、孤独感分、风险等级、最后消息时间。
3. 可用顶部筛选栏按状态 / 渠道过滤，或在搜索框输入昵称后点击 **搜索**。

**期望现象：**

- 列表加载完毕，显示「共 N 条会话 · 第 1 / M 页」。
- 「上一页 / 下一页」按钮在超过 20 条时可用。

---

## 3. 打开会话详情

1. 在会话列表任意一行，点击最右侧 **详情** 按钮。
2. 右侧滑出详情抽屉，展示：用户基本信息、画像评分、最近 50 条消息。
3. 点击抽屉右上角 **✕** 或抽屉外空白区域关闭。

**期望现象：**

- 抽屉内显示昵称、渠道、状态、孤独感分、风险等级、关系阶段等字段（无数据时显示「—」）。
- 消息气泡按时间排列：用户（灰底）、AI（紫底）、运营（橙底）颜色区分。
- 抽屉底部有「查看画像」和「查看记忆」两个快捷链接（若用户 ID 存在）。

---

## 4. 查看用户画像（可选）

方式 A — 从会话列表进入：

1. 在会话列表任意一行，点击 **画像**（蓝色链接）。
2. 跳转至该用户的画像详情页 `/admin/users/{id}`。

方式 B — 从详情抽屉进入：

1. 打开详情抽屉，点击「查看画像」链接。

**画像页期望现象：**

- 顶部显示用户昵称和 user_id（灰色 monospace 字体）。
- 四个分区：基本信息、评分画像（含六条进度条）、偏好与风格、记忆摘要。
- 右上角有「查看记忆 →」链接跳至记忆页。
- 面包屑导航「会话列表 / 用户画像」可点击返回。

**返回会话列表：** 点击面包屑「会话列表」或浏览器后退按钮。

---

## 5. 退出登录

1. 点击顶部右上角 **退出** 按钮。
2. 自动跳转回登录页 `/admin/login`。

**期望现象：** 再次访问 `/admin` 会被重定向到登录页，说明 Token 已清除。

## 常见问题排查

### 白屏 / 页面空白

| 现象 | 最可能原因 | 处理方式 |
|------|-----------|---------|
| 访问 `/admin/login` 完全空白 | 前端静态资源未加载（CDN 或 Nginx 问题） | 刷新（Ctrl+F5）；若持续，联系工程师检查 `https://hugme2.com/health` |
| 登录后跳转到空白页 | 浏览器 localStorage 被清空或 Cookie 限制 | 关闭隐私/无痕模式，用普通浏览器标签页 |
| 画像页 `/admin/users/xxx` 白屏 | user_id 不存在或 API 超时 | 点击面包屑返回，重新从会话列表进入 |

### 401 未授权

| 现象 | 原因 | 处理方式 |
|------|------|---------|
| 登录后立即弹回登录页 | Token 未正确保存（localStorage 被禁用） | 检查浏览器是否开启了「阻止第三方 Cookie」或「严格隐私模式」，切换至正常模式 |
| 操作一段时间后被踢回登录页 | JWT Token 过期（7 天有效期） | 重新登录即可；Token 存在 localStorage，关闭浏览器标签不会失效 |
| 登录时提示「Invalid credentials」 | 用户名 / 密码错误 | 确认密码无前后空格；如忘记密码联系工程师重置 |

### 列表加载失败（红色错误条）

| 现象 | 原因 | 处理方式 |
|------|------|---------|
| 「加载失败：Network Error」 | 浏览器无法连接到服务器 | 检查网络；访问 `https://hugme2.com/health` 确认服务是否正常 |
| 「加载失败：500」或 「加载失败：Service Unavailable」 | 后端服务异常 | 联系工程师，提供出现时间和你的账号名 |
| 「加载失败：401」（列表页） | Token 丢失或过期 | 退出后重新登录 |

### 详情抽屉无法打开 / 一直显示「加载中」

- 网络慢时，详情最多需要 2–3 秒，耐心等待。
- 超过 10 秒仍在加载：点击 ✕ 关闭，重新点击详情按钮再试一次。
- 若反复失败，记录该会话行的用户昵称 + 时间，反馈给工程师。
