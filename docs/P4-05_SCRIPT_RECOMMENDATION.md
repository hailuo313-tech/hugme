# P4-05: 对话区推荐话术列表/一键插入/编辑发送

> **状态**: ✅ 已完成  
> **验收标准**: 坐席可选用库内话术或改字  
> **实现日期**: 2026-05-21

## 概述

本任务在对话详情区域实现了话术库推荐话术功能，坐席可以选用库内话术或改字后发送。

## 实现内容

### 1. 话术库推荐话术显示

#### 状态管理
- 添加 `scriptSuggestions` 状态存储推荐话术列表
- 添加 `scriptLoading` 状态管理加载状态
- 添加 `scriptError` 状态管理错误信息
- 添加 `scriptPanelOpen` 状态控制面板展开/收起

#### 推荐话术获取
- 实现 `loadScriptSuggestions` 函数
- 基于当前对话的用户画像调用 `/scripts/suggest` API
- 支持根据语言、角色、关系阶段、孤独感分等参数匹配
- 在打开对话详情时自动加载推荐话术

#### UI 显示
- 在对话详情抽屉中添加话术库推荐话术面板
- 显示推荐话术列表，最多 5 条
- 显示匹配度百分比
- 显示话术类型信息
- 支持刷新推荐按钮
- 错误处理和空状态显示

### 2. 一键插入功能

#### 填入草稿
- 每条推荐话术都有"填入草稿"按钮
- 点击后将话术内容填入运营回复草稿区域
- 支持后续编辑和修改
- 保留原有草稿内容

#### 编辑功能
- 坐席可以在草稿区域编辑话术内容
- 支持自由修改和调整
- 实时更新草稿状态
- 支持清空草稿

### 3. 发送功能

#### 发送按钮
- 在草稿区域添加发送按钮
- 仅在有草稿内容时显示
- 当前为 TODO 状态，提示需要集成 handoff API
- 绿色按钮，视觉醒目

#### 集成点
- 需要集成 `/api/v1/handoff/{task_id}/reply` API
- 需要传递 conversation_id 和回复内容
- 发送后刷新消息列表

## 技术实现

### API 调用
```javascript
const body = {
  language: conversationData.language || "en",
  loneliness_score: conversationData.loneliness_score || 50,
  risk_level: conversationData.risk_level || "low",
  limit: 5,
  character_id: conversationData.character_id,
  relationship_stage: conversationData.relationship_stage,
};

const resp = await apiFetch("/scripts/suggest", {
  method: "POST",
  body: JSON.stringify(body),
});
```

### UI 布局
- 话术库推荐话术面板位于 AI 辅助面板下方
- 使用天蓝色主题（sky-*）与 AI 辅助的紫色主题区分
- 可折叠设计，默认展开
- 响应式布局，支持移动端

### 交互流程
1. 用户打开对话详情
2. 自动加载话术库推荐话术
3. 用户点击"填入草稿"将话术填入编辑区
4. 用户编辑话术内容
5. 用户点击"发送"发送消息（待集成）

## 使用说明

### 查看推荐话术
1. 在会话列表中点击"详情"按钮
2. 系统自动加载话术库推荐话术
3. 在"话术库推荐话术"面板中查看推荐话术

### 使用推荐话术
1. 点击推荐话术的"填入草稿"按钮
2. 话术内容自动填入运营回复草稿区域
3. 根据需要编辑话术内容
4. 点击"发送"按钮发送消息（待集成）

### 刷新推荐
1. 点击"刷新推荐"按钮
2. 系统重新加载推荐话术
3. 显示最新的匹配结果
